# Requirements Archive: Catat Cuan v2.0 Foundation

**Milestone:** v2.0 Foundation
**Archived:** 2026-05-07
**Status:** Phase 1 requirements validated; Phases 2-6 deferred to next milestone

## Validated Requirements (Phase 1)

### Dark Mode

- [x] **THM-01**: User can select theme mode (Light, Dark, System Default) from Settings with instant switch and no app restart — ✅ Validated v2.0
- [x] **THM-02**: System persists theme preference across sessions using shared_preferences — ✅ Validated v2.0
- [x] **THM-03**: System adapts all UI components for dark mode including glassmorphism containers with adjusted blur and alpha — ✅ Validated v2.0
- [x] **THM-04**: System follows device theme when set to System Default, updating in real-time on platform brightness change — ✅ Validated v2.0
- [x] **THM-05**: Dark theme uses Material Design dark colors with maintained accent colors — ✅ Validated v2.0
- [x] **THM-06**: All text and icons meet accessibility contrast ratios (≥4.5:1 normal text, ≥3:1 large text and icons) — ✅ Validated v2.0

## Deferred Requirements (Phases 2-6)

### Budgeting (Phase 2)

- [ ] **BUD-01**: User can create, read, update, delete monthly budgets per expense category with amount validation
- [ ] **BUD-02**: System tracks spent amount per budget in real-time as transactions are recorded
- [ ] **BUD-03**: User sees visual progress indicators per budget (green/yellow/red)
- [ ] **BUD-04**: User receives in-app alerts when budget reaches thresholds
- [ ] **BUD-05**: User sees budget overview card on Home Screen
- [ ] **BUD-06**: User can view budget vs actual comparison per category
- [ ] **BUD-07**: System enforces one budget per category per month

### Savings Goals (Phase 3)

- [ ] **SAV-01** through **SAV-10**: Goal CRUD, contributions, progress, celebration, home screen card

### Cloud Backup (Phase 4)

- [ ] **BKP-01** through **BKP-07**: Google Drive backup/restore with OAuth

### Enhanced Reports (Phase 5)

- [ ] **RPT-01** through **RPT-07**: Multi-timeframe charts with trend analysis
