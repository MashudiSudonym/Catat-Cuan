### Savings Goal Progress Colors (NEW ‚Äî per SAV-07)

These are **semantic colors** for the goal progress gradient. They apply to circular progress indicators and progress bar fills. Implemented as a helper method in `AppColors` or inline in the progress widget.

| Range | Color | Light Mode Hex | Dark Mode Hex | Usage |
|-------|-------|---------------|---------------|-------|
| 0‚Äì25% | **Red** | `#EF4444` | `#F87171` | Far from goal, critical |
| 25‚Äì50% | **Orange** | `#F97316` | `#FB923C` | Below midpoint |
| 50‚Äì75% | **Yellow** | `#EAB308` | `#FACC15` | Above midpoint, good progress |
| 75‚Äì100% | **Green** | `#10B981` | `#34D399` | Near completion, excellent |

**Implementation approach:**
```dart
static Color getGoalProgressColor(double percentage, bool isDark) {
  if (percentage >= 75) return isDark ? const Color(0xFF34D399) : const Color(0xFF10B981);
  if (percentage >= 50) return isDark ? const Color(0xFFFACC15) : const Color(0xFFEAB308);
  if (percentage >= 25) return isDark ? const Color(0xFFFB923C) : const Color(0xFFF97316);
  return isDark ? const Color(0xFFF87171) : const Color(0xFFEF4444);
}
```
